# Flow

A lightweight desktop environment for kiosks, cyberdecks, and embedded systems.

## Vision

Efficient system management shell that replaces heavy desktop environments with a 30-50MB footprint.

## Branding

- **Name:** Flow  
- **Mascot:** Origami boat (purposeful movement, lightweight, flows with intention)
- **Philosophy:** Smooth, adaptive, effortless interaction

Early development - private repository
